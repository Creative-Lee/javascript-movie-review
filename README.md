## 어플리케이션 실행 방법!

### 1. 패키지 다운받기

```bash
npm i
```

위 명령어로 필요한 패키지를 다운로드해 주세요!

### 2. 프로젝트 실행

```bash
npm run start
```

위 명령어를 터미널에 입력해주세요!

### 3. 영화를 찾아보세요!

```
기본 정렬은 인기순이며, 내가 아는 영화를 검색해볼 수 있습니다!
```
